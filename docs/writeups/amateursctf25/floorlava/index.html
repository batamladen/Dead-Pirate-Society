ABOUT WRITEUPS MEMBERS

Writeup by name

Desafe

Description

dont touch the floor

Simulated 8×8 Grid

The player moves using WASD. Each move toggles a bit in a global 8-byte array byte_4010 using: 

byte_4010[y] ^= 1 << x;

After 28 moves, the program checks each row of byte_4010 against (unsigned char)rand() values from glibc’s RNG with deterministic seeds: 

srand(4919 * i - 559038737);
    if ((unsigned char)rand() != byte_4010[i]) fail;

Output:

e1 e7 a2 d1 b6 e1 ca c4

Bit-Difference Mapping

Each target byte XOR the initial byte produced the mask of bits to flip. Each set bit corresponded to a grid cell that had to be toggled. Total toggles: 28 positions.

Path Reconstruction

The puzzle required a valid sequence of moves on a toroidal 8×8 grid starting from (0,0). A depth-first search found a valid route visiting all required cells exactly once.

Resulting WASD path:

dsddwwawddwddwwddsdddwdwdwwd

Final Decryption

Compute: 

v13 = 0x00ef04f3c3efccc3
    seed = v13 ^ (v13 >> 32) = 0xc300c830

Using glibc’s rand() and XOR on byte_4020 yields plaintext: 

l4va_r3v_05f0d4ff51fb
