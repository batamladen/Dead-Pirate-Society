Space Pirates 2

Description

You are provided with a rust binary.

Walkthrough

The Rust binary is checking for a 32-byte input that, after six reversible transforms, equals the 32-byte TARGET. Because every step is a bijection (XOR, rotations, swaps, modular add/sub, chunk reverse), you can invert them in reverse order on TARGET to recover the required plaintext input.>
