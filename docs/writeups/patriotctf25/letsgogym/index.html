Description

You are given a pcap file and need to find the flag.

Walkthrough

What I noticed when I parsed through the pcap file.

- There are 22 HTTP GET / requests from 192.168.1.235 to 192.168.1.203:8080, using client ports 40000–40021 in order.

- The payload responses are base64 “noise.” The real signal is in the IPv4 header.
- Reading the TTL field from each of those 22 GET / packets, in port order, gives ASCII:
80 67 84 70 123 116 49 109 51 95 116 48 95 103 51 55 95 53 119 48 49 125
